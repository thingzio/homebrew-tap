# thingzio Homebrew Tap

Homebrew tap for [thingzio](https://github.com/thingzio) command-line tools.

## Usage

```sh
brew install thingzio/tap/devradarctl
```

Or tap first, then install:

```sh
brew tap thingzio/tap
brew install devradarctl
```

## Available tools

| Tool | Description |
|------|-------------|
| [`devradarctl`](https://github.com/thingzio/devradarctl) | CLI to generate and submit SBOMs to DevRadar |

## Notes

Casks in this tap are published automatically by each tool's GoReleaser
release pipeline. Do not edit generated cask files by hand.
